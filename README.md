# Countdown Timer

## Project Description
The Countdown Timer is an Android application built using Jetpack Compose and Material3. The app allows users to set countdowns for various activities, helping them keep track of time and stay organized.

## Features
- User-friendly interface
- Easy countdown timer setup
- Notifications for completed counts
- Customizable themes and appearances with Material3

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/loanntt-hn/countdown-timer.git
   ```
2. Open the project in Android Studio.
3. Make sure to have the necessary SDKs installed.
4. Build and run the application on an Android device or emulator.

## Usage Guide
- Launch the app and set your desired countdown duration.
- Tap the start button to begin the countdown.
- You will receive a notification when your countdown reaches zero.

Enjoy staying on schedule with the Countdown Timer app!
